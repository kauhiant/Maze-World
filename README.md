# Maze-World
Unity Version : 2018.2.4f1 

## How to play
- click **./Maze World/Maze World.exe** to start game.
- target : collect all **star** at the maze.
- move : **WASD** or **Arrow key**.
- tools : **mouse** drag to target position.
- pause : press **P**

## 2 Mode
- Maze Mode
  - pure collection.
- Monster Mode
  - avoid monster.
  - use tools to avoid monster.
  - collect all star.
  
## Maze Levels
- unlimited
- add size of maze
